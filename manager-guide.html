Manager Quickstart Guide: Viewing Event Stats

Welcome! This guide helps you quickly access and understand the performance statistics for your events using the QRTick system.

Setting up a new event? Check out our Event Setup Guide for detailed instructions on creating series, performances, and configuring ticket types.

1. Log In

  1. Go to the QRTick login page: https://thing.qrtick.com
  2. Enter the email address and password associated with your manager account.
  3. Click "Sign in".

2. Find Your Event Dashboard

  • After logging in, you'll land on your Dashboard.
  • This page shows cards for all the events you have access to manage or view.

3. Access Event Statistics

  1. Locate the card for the specific event you want to check.
  2. On the event card, find the Stats button (it might be teal-colored).
  3. Click the "Stats" button.

4. Understanding the Stats Page

The Event Stats page gives you a snapshot of performance:

  • Overall Stats: See total tickets sold, how many attendees have checked in, and the check-in percentage.
  • Capacity: View progress towards the event's total capacity (if set).
  • Ticket Types: See a breakdown of sales per ticket type.
  • Recent Orders: View a list of the most recent orders placed, including customer name and number of tickets. (For a full list, click "View All Orders").

5. Orders Page

Need to find specific tickets? The Orders page lets you quickly search by:

  • Customer Name
  • Email Address
  • Phone Number
  • Order ID

For smaller, intimate events, you can use the swipe-to-check-in mode on the Orders page to manually check in attendees without scanning QR codes.

6. Ticket Check-in History

The ticket check-in history feature provides a complete audit trail of all ticket check-in activities:

What It Does

  • See when each ticket was checked in or had a check-in undone
  • Know which staff member performed each action
  • Understand which method was used (QR scan, swipe, etc.)
  • Maintain accountability even when check-ins are reversed

How to Access the History

  1. Go to the event orders screen for any event
  2. Find the ticket you want to review
  3. Click the blue "History" button next to the ticket
  4. View the complete check-in history with timestamps

The history view shows:

  • A summary of the ticket's current status
  • A chronological list of all check-in and undo actions
  • Who performed each action
  • When each action occurred
  • Which method was used for check-in

Benefits for Management

  • Accountability: Know exactly who checked in each ticket and when
  • Dispute Resolution: Easily resolve questions about whether a ticket was previously checked in
  • Security: Identify any unusual or unauthorized check-in patterns
  • Operational Insights: Understand how different check-in methods are being utilized

This feature works automatically in the background with no changes to existing check-in procedures.

7. Quick Access: Add to Home Screen! ✨

For faster access, add the QRTick dashboard directly to your phone's home screen! It will look and feel like a regular app.

On iPhone/iPad (using Safari):

  1. Navigate to your Dashboard page (https://thing.qrtick.com).
  2. Tap the Share button (, usually at the bottom).
  3. Scroll down and tap "Add to Home Screen".
  4. Confirm the name and tap "Add".

On Android (using Chrome):

  1. Navigate to your Dashboard page (https://thing.qrtick.com).
  2. Chrome will automatically prompt you to 'Install app' or 'Add to Home screen'. Look for this prompt near the address bar or bottom of the screen.
  3. Tap the prompt and follow the instructions to install the app to your home screen.
  4. If you accidentally dismiss the prompt, you can usually find the 'Install app' option later in the Chrome menu ().

Now you can launch the dashboard directly from your home screen!

8. Gate Management: POS & Scanners

Efficiently handle at-the-door sales and ticket validation.

Point of Sale (Box Office)

Access the POS via the Sell/Reserve menu entry on your event dashboard.

  • Device: Laptop or tablet recommended.
  • Process: Select ticket type → Enter patron name/phone → Create Sale.
  • Entry: A QR code appears; scan it immediately to mark the patron as entered.

QRTick Scanner Guide

Validate tickets quickly, even offline.

1. Logging In

  • Email/Password: Use your registered credentials.
  • Google: Quick one-tap login.
  • Magic Link: Automatic login via text/email link (no password needed).

2. Selecting Your Event

  • Available Now: Events occurring today.
  • Upcoming: Future events (tickets become scannable 6 hours before start).

3. Scanning Modes

  • Camera Mode (Default): Uses your device's camera.
  • Scanner Device Mode: For professional 2D cordless scanners.
    • Enable in Settings (gear icon) → toggle Scanner Mode.
    • Keeps focus on input for rapid scanning. Camera is disabled to save battery.
Pro Tip: If you've rented one of our scanners, we recommend using Scanner Device Mode. These professional units are significantly faster than using a native phone camera.

4. Scan Results

Result Meaning
VALID Success. Attendee confirmed.
INVALID No such ticket or wrong event.
Already Checked Previously scanned (timestamp shown).
RESERVED Go to Box Office for payment/handling.

5. Offline Scanning

  • Pre-Download: Done automatically when opening the scanner while online.
  • Scan Offline: Validates against local cache.
  • Sync: Auto-syncs when online; check status in Settings.

6. Advanced Settings (Gear Icon)

  • Test Mode: Validates without checking in (great for training).
  • Clear Cache: Forces a fresh ticket data download.
  • Service Worker: Version check and manual updates.
